aaa-visualization-task
======================

Описание
--------

Небольшой проект-описание MVP приложения для подбора недвижимости в Бангкоке для краткосрочной аренды на время отпуска реализованный с помощью Redash.

В `Description.ipynb` собраны:
- описание сценариев и экранов приложения
- примеры визуализаций (распределения, карта, графики)
- SQL-запросы, которые используется для получения данных
